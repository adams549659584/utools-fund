### utools-fund

- 自选基金助手，数据来源于天天基金
- 支持新增删除自选基金，可填入持有份额，实时计算当天收益
- 我的自选基金页面 => ctrl + insert 跳转添加基金页，可根据拼音，中文，基金代码等搜索
- 我的自选基金页面 => ctrl + delete 删除选中的基金
- 我的自选基金页面 => 输入份额，选择基金，enter 保存并更新持有份额
- 我的自选基金页面 => 未输入份额时，enter 打开基金详情
- 基金详情 => enter 关闭， ← → 切换导航，↑ ↓ 切换基金
